# FAB Interactions

Two prototype explorations of a detached FAB on the Cowrywise bottom nav.

**Idea 1 — Tap** opens a quick-actions sheet (Savings, Naira, Dollar Investments).  
**Idea 2 — Hold** fans out investment category pills with a dark overlay.

## Run locally

```bash
# Idea 1
node prototypes/idea-1-quick-actions/server.cjs
# → localhost:8123

# Idea 2
node prototypes/idea-2-long-press-pills/server.cjs
# → localhost:8124
```

Built with static HTML, CSS, and vanilla JS.
